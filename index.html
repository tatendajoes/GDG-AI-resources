About

Welcome to the GDG AI Chatbot learning platform. Here, you'll find resources, tools, and information to help you navigate the project and improve your AI skills.

Weekly Learning Resources

Week 0: Trello Sign Up

Set up Trello accounts and familiarize with the workflow.

Week 1: Getting Started (Supervisor: Tatenda)

Set up GitHub accounts, Fork GitHub Repo, Learn the GitHub Cycle.

Week 2: Sample Chatbot & Web Scraping (Supervisor: Ruvarashe)

Collect links for all resources, Assign and scrape relevant websites/pages.

Week 3: Data Cleaning and Vectorization (Supervisor: Sunday)

Clean and augment data, Vectorize and store processed data.

Week 4: Connect Vector Database and API (Supervisor: Sunday)

Implement connections between the vector database and the API.

Week 5: Testing (Supervisor: Solomon)

Run tests on the system and refine functionalities.

Week 6: Building the Interface (Supervisor: Solomon)

Develop and integrate the user interface.

Week 7: Finalization and Review (Supervisor: Tatenda)

Conduct a project review, Document final insights and improvements.
